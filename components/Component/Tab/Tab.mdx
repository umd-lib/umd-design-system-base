## Tab Component

### Intro

It is the tab component for UMD Libraries. It consists of three subcomponents to display the content in tabs.

### Component Structure

[embed kitchen sink component]

There are three subcomponents to create a working tab component on the page:

- Tabs
- Tab Trigger
- Tab

The Tab Trigger and Tab should be placed under Tabs.

- Tabs
  - Slots - Tab Trigger
  - Slots - Tab

For each Tabs component, you can have as many Tab Triggers and Tabs as needed. It is suggested to have at least two Tab Triggers and Tabs, and no more than five Tab Triggers and Tabs.

The way to connect Tab Trigger and Tab is to use the "tab_value". You should set the "tab_value" of the Tab Trigger and Tab to the same value to connect them. Although you can set the "tab_value" of several Tab Triggers to be the same to show the same Tab, this is not a recommended practice.
