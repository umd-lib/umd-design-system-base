### Implementation Note

The Hero Minimal component is flexible and can be displayed with or without an image. The content width automatically adjusts depending on whether an image is present.

### Field Specifications

<table>
  <tr>
    <td>Field</td>

    <td>Note</td>
  </tr>
  <tr>
    <td>Image</td>

  <tr>

### Twig Template

```twig
<div class="hero--minimal c-bg-primary c-content-primary">
  {% if image %}
  <div class="hero--image">
    <img alt="{{ image.alt }}" src="{{ image.url }}">
  </div>
  {% endif %}
  <div class="hero--container s-box-page-default-h s-box-page-default-v s-center">
    <div class="hero--content s-box-large-h {% if not image %}text-only{% endif %}">
     {% if eyebrow %}
        <p class="hero--eyebrow c-content-secondary t-eyebrow s-stack-small">
          {{ eyebrow }}
        </p>
      {% endif %}
      <h1 class="hero--headline t-display s-stack-default">
        {{ title }}
      </h1>
      {% if description %}
        <p class="hero--description c-content-secondary t-body-medium">
          {{ description }}
        </p>
      {% endif %}
    </div>
  </div>
</div>
```
